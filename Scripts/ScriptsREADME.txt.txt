SCRIPTS USAGE GUIDE:

1. run_test.bat
   - Runs all performance tests sequentially
   - Generates .jtl result files and HTML reports
   - Update JMETER_PATH variable if JMeter is installed elsewhere

2. generate_report.bat
   - Generates HTML reports from existing .jtl files
   - Useful for re-generating reports without re-running tests

PREREQUISITES:
- Apache JMeter 5.6.3+ installed
- Java 8+ installed
- Update JMETER_PATH in scripts to match your installation

NOTE: Remove or modify credentials in test plans before sharing publicly!
